域名购买

什么是域名 domain names

域名 Domain names：互联网上用于 标识 和 定位特定网站 或 资源 的人类可读的文字标识。

简单来说，我们使用的网址都是域名。如：www.example.com 就是一个域名。

在互联网中，知道了一台电脑的 IP 就可以访问该电脑上的资源，但是纯数字的 ip，对于人来说太难记住了，所以就出现了 域名 Domain Names，

将 域名 和i p地址 对应起来就可以通过 域名 来 访问对应的服务器。

域名 可以将人类友好的文字与互联网上的 IP地址 关联起来。

简单来说，域名类似于人名，谁都可以叫这个名字，即人名不能指代具体的人，域名也是一样，不能真正的指代某一个服务器，域名谁买了/注册了就有了一段时间的使用权（取决于购买的时间）。

ip地址就是身份证号，只有身份证号码，才能唯一标识一个人；也只有IP地址才能真正指代一个网站的服务器（虽然ip地址也可能会发生改变）。

通过将 ip地址（公网ip） 和 域名 进行绑定就能通过域名访问ip地址对应的电脑了。

简单的网页访问原理

当你在浏览器中输入一个域名时，计算机会通过域名系统（Domain Name System，DNS）将该域名解析为对应的IP地址，然后将你的请求发送到正确的服务器，从而让你能够访问到该网站或资源。

域名的组成

一般 域名 由两到三个部分组成（有可能只有一部分，当然也有更长的）的简单结构，它被点分隔。 域名是从右往左看的

image.png

顶级域名 TLD

顶级域名（Top-Level Domain，TLD）：通常用于表示域名的最高级别部分，即一个域名最右边的部分

顶级域名可以告诉用户域名所提供的服务类型（但也没有强制），比如 .com (商业),.org（非盈利组织）、.net（网络）、.gov（政府）、.edu（教育）等。

在 blog.example.com 中 .com 就是顶级域名

注意

  1. 一般来说，顶级域名和顶级域是一个概念。这里说的域不同于计网中的域。

  2. 由于关于域名的各种资料水平参差不齐，暂时还没找到权威性的资料，所以对于一些概念可能会存在一些争议。

  3. 有很多介绍域名的文章中没有介绍一级域名，有些文章写一级域名就是顶级域名，还有些文章中一级域名和二级域名的概念一致

  4. 还有一些文章中二级域名和子域名的概念一致，所以具体怎么看见仁见智，本文只代表个人看法。

一级域名 FLD

一级域名（First-Level Domain，FLD）：一级域名是指在域名中的主体部分，通常是顶级域名之前的第一部分。

一级域名在你注册或者购买域名时就已经确定了。因为一般购买的都是一级域名，所以无法自行更改，只能重新注册/购买一个新的域名。

在 blog.example.com 中example就是一级域名。而对于blog.example.com这整个域名来说，是一个二级域名。而对于example.com这整个域名来说是一个一级域名

二级域名 SLD

二级域名（Second-Level Domain，SLD）：二级域名是位于一级域名之前的部分，用于进一步分类和组织网站的不同部分或服务。

它是可自定义的，比如你购买了一个域名 example.com ，你可以在很多域名注册商提供的 DNS 中自行设置，如：blog.example.com 、www.example.com

在 blog.example.com 中 example 就是二级域名。

子域名 Subdomain

子域名（Subdomain）：子域名是可以选择性添加到顶级域名、一级域名或二级域名 之前的部分，用于进一步分类和组织网站的不同部分或服务。

子域名也是域名，只是一个相当概念，它是上一级域名的一部分。

在 blog.example.com 中 blog.example.com 是 example.com 的子域名；当然 example.com 也是 .com 的子域名。

购买域名

选择合适域名服务提供商，在每一个地方比比价，哪一个便宜选哪一个。常见的域名服务提供商：阿里云、腾讯云、百度云、couldfare、NameSilo，Dynadot等、价格合适就行。一般推荐国内的直接购买、因为付款方便。

域名购买前要注意哪些？

  1. 一般购买域名时，第一年会比较便宜，但是，后面续费的价格比较高，所以在购买时要多注意一下续费的价格。

  2. 在国内的一些域名服务提供商购买域名，需要个人实名认证，信息详情：姓名、地址、电话、邮箱、邮编、彩色身份证图片。（国外的不太清楚，应该不需要个人的省份证号吧）

  3. 上面说了只有域名是无法进行访问对应的网站的，需要将 域名 和 ip 进行绑定之后，才能访问网站，而这个 ip 是在购买云服务器会提供、或者家里的网络本身就是 公网ip 才行，所以一般 域名 要搭配 服务器 才能正常使用。

  4. 在国内购买域名需要实名认证，且使用国内的服务器，则需要对网站进行备案后，网站才能正常使用

  5. 使用国内购买的域名，使用国外的服务器，则不用进行备案，购买就可以直接使用。

  6. 在国外购买的域名，就只能使用国外或香港主机/服务器，不能解析到国内的服务器。

  7. 在国外购买的域名，想要使用国内的云服务器，必须先将 域名 转移到国内的域名服务提供商，才能使用国内的服务器。

  8. 自己参考自己建站的需求选择对应的域名服务提供商进行域名购买。

阿里云域名购买

为什么使用在阿里云购买域名？

因为 便宜、方便、阿里云时间久，使用的人群多，国内头部、至少不会说跑路就跑路。

  1. 拥有一个 支付宝账号 ，在阿里云官网进行登录

  2. 点击右上角 控制台；

image-2.png
  1. 鼠标移到左上角，在搜索栏中搜索 域名；
image-3.png
  1. 点击 域名 ，进入 域名控制台；
image-4.png
  1. 在左侧导航栏单击 信息模板；；
image-5.png

  1. 创建信息模板，完成信息的填写、提交；耐心等待注册局审核中，大概几个小时内完成审核。在此期间，可以先想一想要注册的域名，审核完成后、会有短信通知。

  2. 然后还是在左侧导航栏单击 概况，搜索自己喜欢的域名，

image-7.png
  1. 选择合适 加入域名清单，点击 域名清单，点击立即购买，选择自己的信息模板，然后 点击立即购买，完成支付即可。
